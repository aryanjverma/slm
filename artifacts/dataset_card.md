# Dataset Card: Arithmetic Tutor SLM

## Purpose

Synthetic supervised fine-tuning data for a Socratic addition/subtraction tutor. The assistant should help with the next arithmetic step without revealing the final answer.

## Files

- `artifacts/data/train_cases.jsonl`: structured v1 training cases.
- `artifacts/data/train_chat.jsonl`: chat-format v1 SFT rows.
- `artifacts/data/eval_cases.jsonl`: held-out evaluation cases.
- `artifacts/data/train_cases_v2.jsonl`: targeted v2 structured cases.
- `artifacts/data/train_chat_v2.jsonl`: targeted v2 chat-format SFT rows.

## Generation

Data is generated by `src/arithmetic_tutor_slm/data.py` with deterministic arithmetic ground truth. Each row includes the problem, student state, hidden final answer, expected next step, mistake type, difficulty, tags, and target assistant response.

## Filtering

Rows are rejected when the assistant response leaks the final answer, contains too many final-answer digits, is too long, or does not behave like a Socratic hint.

## Intended Use

Use `train_chat.jsonl` for v1 SFT. Use `train_chat_v2.jsonl` for a second run or merge it with v1 to oversample direct-answer pressure, alignment, wrong-answer checking, and borrow-through-zero cases.
