# APUSH LEQ Grader Public Companion

> This is a redistribution-safe companion built from project-authored synthetic data. It is not the private corpus used for final v5 training.

This repository demonstrates the FRQCase schema, structured grading target, and public synthetic baseline data. It contains no private v5 essays, labels, style references, review packets, or predictions. See `manifest.json` for hashes and counts.
